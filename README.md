# CAlCULATOR

## A simple calculator to perform basic algebraic operations using two numbers.

## Operations
- [x] Addition (+)
- [x] Subtraction (-)
- [x] Multiplication (*)
- [x] Division (/)
- [x] Square (**)


## Usage
- Ensure you have python3 installed.
- run the program `calculator.py`.
- Enter first number.
- Enter second number.
- Enter mathematic operator.
- Boom! you have your answer.
